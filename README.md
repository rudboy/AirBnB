- Display appartement with information
- click on it for display map marker in Google map
- Navigate on BottomVavigator
- Log out

## Running the project

Clone this repository :

```bash
git https://github.com/rudboy/AirBnB

cd AirBnB
```

Start the server:

```bash
npm start
```

Build the project

```bash
npm run build
```
